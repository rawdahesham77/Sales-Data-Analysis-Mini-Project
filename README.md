🟣 Sales Data Analysis (Python)
✨ Description

This project performs exploratory data analysis (EDA) on a sales dataset using Python. The goal is to clean the data, analyze sales performance, and extract meaningful insights to support business decisions.

📊 Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook
📁 Dataset

The dataset contains sales transactions including product, quantity, price, and date information.

🔧 Data Cleaning
Removed missing values
Converted data types (dates, numbers)
Fixed inconsistent data entries
Created new columns for analysis
📈 Analysis

The analysis answers key business questions such as:

What is the best month for sales?
Which products are the most sold?
What time do customers buy the most?
Which cities generate the highest revenue?
🔍 Key Insights
Identified the highest revenue month
Found top-performing products
Detected peak sales hours
Highlighted best-performing locations
📊 Visualizations
Sales by Month
Sales by City
Product Performance
Time-based Analysis
💡 Recommendations
Increase stock for top-selling products
Focus marketing campaigns during peak hours
Expand in high-performing cities

🎯 Conclusion
This analysis helps understand sales trends and provides insights that can improve business strategies and increase revenue.
